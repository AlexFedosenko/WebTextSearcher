Данное приложение предназначено для распаралелленного поиска текста в контенте web-страниц. Если в процессе поиска на странице были найдены еще URL, то такие ссылки добавляются в список для будущего сканирования. Длина такого списка ограничивается введенным пользователем значением.

Перед началом работы пользователь должен указать 
1) URL исходной страницы;
2) Искомый текст;
3) Количество потоков поиска;
4) Количество URL для сканирования.

Также доступна опция поиска информации и новых URL в метаданных вебстраницы. 

Статус сканирования, текущий прогресс и список найденных URL отображаются на вкладке PROGRESS. Все результаты (найденные совпадения с искомым текстом) отображаются на вкладке RESULTS. Если были найдены новые результаты, данная вкладка помечается спец. симоволом "*".

Также в приложении доступна опция паузы и полной остановки сканирования. Приложение адекватно реагирует на переворот экрана.
